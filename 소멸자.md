# 소멸자

객체소멸시 소멸자가 호출된다 특징은 다음과 같다. </br>
> 클래스 이름 앞에 ~가 붙는다. </br>
반환하지 않는다. </br>
오버로딩도 디폴트 값도 설정 불가 </br>

소멸자는 자동으로 호출된다. </br>
프로그래머가 직접 소멸자를 설정하지 않으면 디폴트로 </br>
설정된다. 즉 다음과 같다. </br>
```c++
class Genius{
  ///
};
```
</br>

```c++
class Genius{
  Genius(){}
  ~Genius(){}
};
```
</br>
위 두 코드는 똑같다. </br>
소멸자는 생성자내에서 할당한 메모리공간을 해체하는데 사용된다. </br>
