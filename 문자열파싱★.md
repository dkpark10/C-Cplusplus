# 문자열 파싱 

문자열을 다루는건 은근히 정말 어렵고 짜증난다. </br>
입력으로 다음과 같이 들어올 때</br>

> 123 235234 32 88 909

```c++
string s
vector<int>v;
getline(cin,s);
int begin = 0;
for(int i=0; i<s.size(); i++){
  if(s[i] == ' '){
    v.push_back(stoi(s.substr(begin, i - begin)));
  }
}
v.push_back(stoi(s.substr(begin)));
```

벡터에 숫자를 차례대로 담을 수 있다.
